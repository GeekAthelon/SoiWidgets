The Map Editor Software for SOI

The boring stuff

Right now, this program is still somewhat experimental and NOT ready for prime time use.

Permission to use this map program AT SOI is granted, so long as my name remains intact.

How it works

The map viewer works by taking a regular image file with no writing on it, and then reads through a list of names and places, and writes various legends on the map. In theory, this makes updating the map very easy compared to the older method of actually having to change the graphic.

There are several parts of map viewer system, and all of them must be in place.

makeseebutton.js
This file must be included on any page that will link to the map page.
mapedit.html
This is the image map viewer and editor
map_show.js
The driving force behind mapedit.html

How to install it

These directions assume you are fairly comfortable with HTML and basic file editing.

Upload the following files to whichever room will be hosting the map viewer

  1. makeseebutton.js
  2. map_show.js
  3. mapedit.html

Edit madeseebutton.js and edit the variable URL_TO_MAP_PAGE to refer to the mapedit.html that you just uploaded.

In any room that will link to the map file, go to room controls and make the following two changes:

  1. Add
    <script src="PATH_TO_YOUR_ROOM/makeseebutton.js"></script>
    It doesn't matter where you add it.
  2. Whereever you would like a link to the map file, add this code:
    <span id="putSeeMapHere">SeeMapWillGoHere2</span>
    You may only add this once per room.

Once you have the map viewer/editor installed, go the room that has the link to the map viewer and hit 'view map.' You will see a rather basic map with just a few entries on it.

I'm not going to detail how to use the editor here, it should be fairly simple to work out. Once you have your new rooms areas added, click on the Generate Javascript and then cut-and-paste the given code into the file map_show.js, just overwriting the data that is there -- NOT the entire file.
