Setting up User Flairs

Step 1 - Add the JavaScript to your room

Under Room Controls, add this code to the "Text for Bottom of Room" section:

<script src="http://soiroom.hyperchat.com/chatplus/flair.js"></script>

Step 2 - Configure Nicknames and Flair Messages

Apparently some rooms mangle the JavaScript when it is embedded in the page. I now recommend creating a separate file.

Go to the FTP Files for the room you are working with and create a new file called flair-config.js. Note, you'll be able to edit the file the same way, by typing flair-config.js into the Create filename box and pressing the button.

You'll need the full URL of this file and you'll need to include a new script element in your page setup to load this.

For example, in my case the URL to the file is:
http://soiroom.hyperchat.com/testdice/flair-config.js

So I would include the line
<script src="http://soiroom.hyperchat.com/testdice/flair-config.js"></script>

The format is fairly simple:

[NickName, FlairMessage, CSSClass]

The nickname must be exactly as it appears in the room. Note that while my nickname is "Athelon", with the formatting it appears as "-=[Athelon]=-".


var flairs = [
  ["-=[Athelon]=-", "Athelon Da Geek", "ath"],
  ["-=[Lord of the Dice]=-", "Noble User of Dice", "ld"],
  ["Visitor", "Honored Visitor", "vi"],
];

Step 3 - Add the basic CSS

While this step is optional, it is strongly recommended that you add the basic CSS to make the Flair messages stand out a bit more and be noticed.

Go to the FTP_File Controls and create the file flair-style.css by typing the name
flair-style.css into the Create File box and pressing the Edit File button. You can later EDIT the file the same way.

Give the file this content:

.all-flairs {
  display: inline-block;
  margin-left: 2.5em;
  padding-left: 1em;
  padding-right: 1em;
  border-radius: 25px;

  border: 1px dotted white;
  border-style: outset;
  background-color: black;
  color: silver;
}

.all-flairs > span {
  font-size: 75%;
}

Next you will need the URL of that file. In my case the file name is:

http://soiroom.hyperchat.com/testdice/flair-style.css

So I will go to my Room Controls and add the following line to the bottom of the page controls.

<link rel="stylesheet"
 type="text/css" 
 href="http://soiroom.hyperchat.com/testdice/flair-style.css">

Step 4 - Add Optional Custom CSS

You may wish to give different people, or different groups of people, different colored flairs. You may wish to give close friends one color of flair, while giving frequent visitors another. This is where that third option in the flairs set up comes in. In my example, I used the names of ath, ld and vi.

To give those their own coloring, simply add this code to the end of the flair-style.css created in step 3.

.ath {
  border: 2px solid yellow;
  border-style: outset;
  background-color: black;
  color: yellow;
}

.ld {
  border: 2px solid black;
  border-style: outset;
  background-color: white;
  color: black;
}


.vi {
  background-color: solid blue;
  border-style: outset;
  color: yellow;
}
