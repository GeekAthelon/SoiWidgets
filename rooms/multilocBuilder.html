Step 2: Add, remove, or change a view.

To create a new view, select the Add radio button, enter the room id (2-8 alphanumeric characters) in the text box, and press "Make Change".

If you wish to delete a view, select the Delete radio button, enter the room id you wish to delete in the text box, and press "Make Change".

There is no maximum number of views you can create, but over a dozen become awkward.
